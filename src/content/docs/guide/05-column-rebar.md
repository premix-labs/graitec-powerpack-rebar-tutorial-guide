## 8. แบบฝึกหัดที่ 3: ใส่เหล็กเสาด้วย Main Bars

เสาจะใช้วิธีคล้ายคาน แต่ต้องดูตำแหน่งเหล็กในหน้าตัดเสา

> หมายเหตุ: ค่า `4-13M`, `10M @150`, offset, และจำนวนเหล็กในบทนี้เป็นค่าฝึกเพื่อให้เห็นผลใน `Section` และ `3D` ได้ชัด ไม่ใช่ค่าที่ต้องใช้เหมือนกันทุกโครงการ

### 8.1 เตรียมมุมมอง

1. เปิด `3D View`
2. เปิด `Section` หรือมุมมองที่เห็นเสาชัด
3. ตั้ง `Detail Level = Fine`

### 8.2 เลือกเสา

1. คลิกเลือกเสา 1 ต้น
2. ดู `Properties`
3. ยืนยันว่าเป็น `Structural Column`

### 8.3 เรียกคำสั่ง Main Bars

1. ไปที่แท็บ `PowerPack Rebar`
2. คลิก `Main Bars`

### 8.4 กำหนดค่า

ในบทเสานี้ ให้ใช้ค่าตั้งต้นสำหรับเสาตัวอย่างแบบง่ายก่อน เพื่อให้เห็นผลชัดและตรวจใน section ได้ง่าย

### 8.4.1 ค่า `Main Reinforcement` ที่ให้กรอก

เมื่อเข้าหน้า `Longitudinal reinforcement > Main Reinforcement > Main Bars` ให้กรอกดังนี้

- `Quantity along a` = `2`
- `Quantity along b` = `2`
- `Rebar type along x` = `13M`
- `Rebar type along y` = `13M`
- `Bottom offset` = `0 mm`
- `Top offset` = `0 mm`
- `Automatic hook lengths` = เปิดไว้
- `Top hook angle` = `0°`
- `Bottom hook angle` = `0°`

ค่าชุดนี้จะได้เหล็กหลัก 4 เส้นที่มุมเสา

### 8.4.2 วิธีเช็กว่ากรอกถูก

ถ้ากรอกถูก คุณควรเห็นลักษณะดังนี้ในหน้าต่างคำสั่ง

- จำนวนเหล็กรวมเป็น 4 เส้น
- รูปตัวอย่างทางขวาแสดงเหล็กที่มุมเสา
- ยังไม่จำเป็นต้องมีปลอกในขั้นนี้

จากนั้นกด `OK`

### 8.4.3 ถ้าจะเพิ่ม Secondary Bars

ถ้าเสาต้นที่กำลังฝึกมีหน้าตัดค่อนข้างใหญ่ หรือคุณต้องการให้เสามีเหล็กยืนมากกว่า 4 มุม ให้กลับเข้า `Main Bars` อีกครั้ง แล้วไปที่ `Longitudinal reinforcement > Main Reinforcement > Secondary Bars`

สำหรับการทดลองครั้งแรก ให้ใช้ค่าตั้งต้นแบบง่ายก่อนดังนี้

- `Quantity along a` = `1`
- `Quantity along b` = `0`
- `Rebar type along x` = `13M`
- `Rebar type along y` = `13M`
- `Bottom offset` = `0 mm`
- `Top offset` = `0 mm`

ความหมายของค่าชุดนี้คือ

- เพิ่มเหล็กยืนที่ด้านยาวของเสาอีกด้านละ 1 เส้น
- ทำให้จากเดิมที่มีเฉพาะเหล็กมุม 4 เส้น เริ่มมีเหล็กกลางด้านเพิ่มขึ้น
- ยังเหมาะกับการฝึกดูผลใน `Section` และ `3D` ได้ง่าย

สิ่งที่ควรเช็กหลังกรอก

- รูปตัวอย่างทางขวาต้องมีเหล็กเพิ่มจาก 4 มุมเดิม
- เหล็กใหม่ต้องยังอยู่ภายในหน้าตัดเสา
- ถ้ารูปดูแน่นเกินไป ให้ลดจำนวนหรือใช้เหล็กขนาดเล็กลงชั่วคราว

หมายเหตุ:

- `Secondary Bars` คือเหล็กยืนเสริม ไม่ใช่เหล็กปลอก
- ถ้าเพิ่งเริ่มฝึกและยังต้องการเสาแบบง่ายที่สุด สามารถข้ามหัวข้อนี้ไปก่อนได้
- ถ้าเสาในโปรเจกต์จริงมีเหล็กยืนหลายเส้นต่อด้าน มักต้องใช้หัวข้อนี้

### 8.4.4 ถ้าจะเพิ่ม Cross Tie

ถ้าเสามีเหล็กยืนเพิ่มจาก `Secondary Bars` แล้ว หรือหน้าตัดเสาเริ่มใหญ่จนปลอกนอกอย่างเดียวรัดเหล็กด้านในไม่พอ ให้ไปที่ `Transverse Reinforcement` แล้วมองหาตัวเลือกที่เกี่ยวกับ `Cross Tie`, `Along a`, `Along b` หรือการรัดเพิ่มด้านในของปลอก

แนวคิดของ `Cross Tie` คือ

- เป็นเหล็กรัดเพิ่มด้านในของเสา
- ใช้ช่วยรัดเหล็กยืนที่ไม่ได้อยู่เฉพาะมุมเสา
- มักใช้เมื่อเสามีเหล็กยืนหลายเส้นต่อด้าน หรือเสาหน้าตัดใหญ่

สำหรับการฝึกครั้งแรก ให้ลองแบบง่ายที่สุดก่อนดังนี้

- ใส่ `Secondary Bars` ให้เกิดเหล็กกลางด้านก่อน
- กลับเข้า `Transverse Reinforcement`
- ถ้ามีช่อง `Along a` หรือ `Along b` ให้ลองเลือกด้านที่มีเหล็กกลางด้านเพิ่ม
- ถ้ามีตัวเลือกจำนวน cross tie ให้เริ่มจาก `1`
- ถ้ามีตัวเลือกชนิดเหล็ก ให้ใช้ชนิดเดียวกับปลอกหลักก่อน เช่น `10M`

สิ่งที่ควรเช็กหลังเพิ่ม

- ในรูปตัวอย่างทางขวาต้องเห็นมีเหล็กรัดด้านในเพิ่ม ไม่ใช่มีแค่ปลอกรอบนอก
- เหล็กยืนกลางด้านควรถูกรัดโดยปลอกหรือ cross tie อย่างชัดเจน
- ถ้ารูปดูแน่นเกินไป ให้ลดจำนวน cross tie ลงก่อน

หมายเหตุ:

- บางไฟล์หรือบางเวอร์ชันของคำสั่ง อาจไม่ได้ใช้ชื่อ `Cross Tie` ตรง ๆ
- ถ้าไม่เห็นคำนี้ ให้มองหาช่องที่ใช้กำหนดปลอกรัดเพิ่มตามแนว `a` และ `b`
- สำหรับเสาฝึกพื้นฐานที่มีแค่เหล็กมุม 4 เส้น ยังไม่จำเป็นต้องใส่ cross tie ทุกต้น

### 8.4.5 ถ้าจะใส่ปลอกเสาต่อ

ถ้าต้องการให้เสาครบทั้งเหล็กหลักและปลอก ให้กลับเข้า `Main Bars` อีกครั้ง แล้วไปที่ `Transverse Reinforcement`

ค่าตั้งต้นที่ใช้แล้วขึ้นจริงในตัวอย่างนี้มีดังนี้

- `Rebar type` = `10M`
- `Configurations` = `Steel Binders`
- `Along a` = `None`
- `Along b` = `None`
- `Define distribution by` = `Quantity`
- `Distribution` = `Custom`
- `Bottom offset (O1)` = `50 mm`
- `Top offset (O2)` = `50 mm`

จากนั้นกด `+` ในตารางด้านล่าง แล้วกรอกค่าของ package แถวแรกดังนี้

- `Q` = `19`
- `S` = `150 mm`
- `L` = ให้ระบบคำนวณเอง ถ้าช่องแก้ไม่ได้

หมายเหตุเพิ่มเติม:

- ในตัวอย่างเสาสูง `3000 mm` โปรแกรมอาจคำนวณ `L` เป็นประมาณ `2850 mm`
- ถ้าช่อง `L` แก้ไม่ได้ ถือว่าปกติ ให้ปล่อยตามที่ระบบคำนวณ
- หลังจากกด `OK` ควรเห็นปลอกเสาใน `3D` และ `Section`

หมายเหตุ:

- ในบางโปรเจกต์ ชื่อช่องอาจไม่ตรงกันทุกตัว แต่ให้มองหาค่าประเภทเดียวกัน
- ถ้าใส่ปลอกแล้วไม่เห็นใน `Plan` ไม่ได้แปลว่าไม่มีปลอก ให้เช็กใน `Section` เป็นหลัก

### 8.5 เช็กผลใน 3D และ Section

1. คลิกกลับไปที่ `3D View`
2. ซูมเข้าใกล้เสา
3. ถ้ายังดูยาก ให้เปิด `Section`
4. ดูว่าเหล็กอยู่ในหน้าตัดเสา

### สิ่งที่ควรเห็นถ้าทำถูก

- เหล็กอยู่ภายในเสา
- ตำแหน่งดูสมมาตร
- ไม่มีเหล็กยื่นพ้นคอนกรีต

### ถ้ามีข้อความเตือนว่าอยู่นอก host

ให้ทำตามนี้

1. คลิกเลือกเสา
2. ดูค่า cover ใน `Properties`
3. ลดขนาดเหล็กหรือจำนวนเหล็กชั่วคราว
4. ทดลองคำสั่งใหม่อีกครั้ง
