## 93. บทนี้ใช้ตอนไหน

ใช้บทนี้เมื่อคุณเจอกรณีต่อไปนี้

- ชื่อเมนูใน PowerPack หรือ Revit ไม่ตรงกับในหนังสือ
- ไม่มี `Rebar type` แบบ `10M` ตามตัวอย่าง
- ไม่มี `Rebar Cover` แบบ `40 mm` หรือ `50 mm`
- ยังไม่มี `Section` หรือ `3D View` ที่พร้อมตรวจผล
- คำสั่งเปิดได้ แต่ field บางตัวไม่เหมือนกับตัวอย่างในหนังสือ

บทนี้ไม่ได้แทนบทฝึกหลัก แต่ช่วยตัดสินใจเวลาสภาพแวดล้อมของคุณไม่เหมือนตัวอย่าง

---

## 94. ถ้าชื่อเมนูไม่ตรงกับหนังสือ

ให้ใช้หลักนี้แทนการหาชื่อให้ตรงทุกคำ

1. หา command หลักให้ถูกก่อน เช่น `Main Bars`, `Copy Rebar`, `Openings`
2. ดู preview ของหน้าต่างประกอบเสมอ
3. เทียบจากหน้าที่ของ field แทนชื่อ เช่น field ของ `Spacing`, `Offset`, `Top`, `Bottom`
4. ถ้าต้องเลือกระหว่างสองเมนู ให้สร้างทีละชั้นหรือทีละชุด แล้วเช็กใน `Section`

> [!IMPORTANT]
> ถ้าชื่อเมนูไม่ตรง อย่าทำหลายขั้นพร้อมกัน ให้ทำทีละชุดและใช้ `Section` เป็นตัวตัดสิน

---

## 95. ถ้าไม่มี Rebar Type ตามตัวอย่าง

ตัวอย่างในเล่มมักใช้ `10M` เพื่อให้ฝึก workflow ได้ง่าย แต่ถ้าในไฟล์ของคุณไม่มี ให้ทำดังนี้

1. เลือกขนาดที่เล็กและใกล้เคียงที่สุดที่มีอยู่
2. คง logic เดิมของบท เช่น `Spacing`, `Offset`, `Hook`
3. เช็กใน `3D` และ `Section` ว่าเหล็กยังอยู่ใน host
4. ถ้าขนาดใหม่ใหญ่ขึ้นมาก ให้เช็ก `Rebar Cover` ซ้ำ

ห้ามทำสิ่งนี้

- อย่าเปลี่ยนทั้ง `Rebar type`, `Spacing`, และ `Offset` พร้อมกันถ้ายังหาเหตุไม่เจอ

---

## 96. ถ้าไม่มี Rebar Cover ตามตัวอย่าง

ถ้าใน dropdown ไม่มี `40 mm` หรือ `50 mm` ให้ทำลำดับนี้

1. ไปที่ `Manage`
2. หาเมนูที่ใช้จัดการ `Rebar Cover Settings`
3. สร้าง cover type ใหม่ให้ตรงกับค่าที่ต้องการ
4. กลับมา assign `Rebar Cover` ให้ host ก่อนสร้างเหล็ก
5. ค่อยตั้ง offset ให้สอดคล้องกับ cover

> [!WARNING]
> อย่าฝืนแก้ปัญหาด้วยการใส่ offset แปลก ๆ ทั้งที่ `Rebar Cover` ของ host ยังไม่ตรง เพราะจะทำให้ debug ยากในบทถัดไป

---

## 97. ถ้ายังไม่มี Section หรือ 3D View

ก่อนฝึกงาน `Wall`, `Footing`, `Openings`, `Detail`, หรือ `Clone Drawings` ให้เตรียมอย่างน้อย

1. `3D View`
2. `Section` ที่ตัดผ่าน host

ถ้ายังไม่มี

1. ไปที่แท็บ `View`
2. ใช้คำสั่ง `Section`
3. ลากผ่าน host ที่ต้องการ
4. เปิดมุมมองนั้นทันที

หลักจำง่าย

- `3D View` ใช้ดูภาพรวม
- `Section` ใช้ตัดสินว่าเหล็กอยู่ใน host จริงหรือไม่

---

## 98. ถ้าใช้คนละเวอร์ชันกับหนังสือ

สิ่งที่มักต่างกันคือ

- ชื่อ field
- ตำแหน่งปุ่มย่อย
- preview ของหน้าต่าง
- พฤติกรรมบางคำสั่งหลังคำนวณ

สิ่งที่ไม่ควรเปลี่ยนแม้ต่างเวอร์ชัน

1. ต้องเลือก host ให้ถูก
2. ต้องตั้ง `Rebar Cover`
3. ต้องตรวจ `Detail Level = Fine`
4. ต้องเช็กใน `3D` และ `Section`
5. ต้องสร้างทีละชุดเมื่อยังไม่แน่ใจ

---

## 99. ลำดับตัดสินใจที่ปลอดภัยที่สุด

1. เช็ก host
2. เช็ก `Rebar Cover`
3. เช็ก `3D View` และ `Section`
4. ใช้ขนาดเหล็กใกล้เคียงที่สุดที่มี
5. สร้างทีละชุด
6. เช็กผลทันที
7. ค่อยเพิ่มชั้น เพิ่ม copy หรือออกเอกสาร

---

## 100. สรุปแบบสั้นที่สุด

- ชื่อเมนูต่างได้ แต่หลักการ `host -> cover -> view -> create -> check` ต้องเหมือนเดิม
- ถ้าไม่มี type หรือ cover ตามตัวอย่าง ให้แก้ที่สภาพแวดล้อมก่อน ไม่ใช่ไปฝืนค่าที่บทสอน
- ถ้ายังไม่แน่ใจ ให้ทำทีละชุด แล้วตัดสินด้วย `Section`
