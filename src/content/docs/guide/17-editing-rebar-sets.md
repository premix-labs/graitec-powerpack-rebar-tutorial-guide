## 61. ทำไมต้องมีบทนี้

ในงานจริง เราไม่ได้สร้างเหล็กแล้วจบในครั้งเดียวเสมอไป  
บ่อยครั้งต้องตัด แยก ลบ หรือจัดชุดใหม่หลังจากสร้างไปแล้ว

บทนี้จึงเน้นเครื่องมือกลุ่มแก้ไข

- `Delete Rebar`
- `Split Rebar`
- `Cut Rebar`
- `Rebar Set`

---

## 62. Delete Rebar

ใช้เมื่อต้องการลบเหล็กที่สร้างจาก PowerPack ออกแล้วเริ่มใหม่

เหมาะกับกรณี

- ค่าที่ใส่ผิดตั้งแต่ต้น
- รูปแบบเหล็กไม่ตรงกับที่ต้องการ
- ต้องการเคลียร์โมเดลก่อนลองอีกครั้ง

### วิธีใช้แบบเร็ว

1. เลือก host หรือชุดเหล็กที่ต้องการลบ
2. ไปที่แท็บ `PowerPack Rebar`
3. คลิก `Delete Rebar`
4. ตรวจใน `3D` ว่าชุดที่ผิดหายไปจริง
5. ถ้าเตรียมจะสร้างใหม่ ให้เช็กก่อนว่าลบเฉพาะชุดที่ตั้งใจ ไม่ใช่ลบทั้ง host โดยไม่รู้ตัว

---

## 63. Split Rebar

ใช้เมื่อต้องการแยกเหล็กหรือแบ่งชุดเหล็กออกเป็นช่วง

เหมาะกับกรณี

- ต้องการแยกเหล็กเป็นหลายช่วง
- ต้องการแก้เฉพาะบางส่วนของชุดเหล็ก
- ต้องการควบคุม detailing ให้ละเอียดขึ้น

### วิธีใช้แบบเร็ว

1. เลือกชุดเหล็กที่สร้างแล้ว
2. คลิก `Split Rebar`
3. ระบุตำแหน่งหรือหลักการที่ต้องการแบ่ง
4. กด `OK` หรือ `Apply` ตามหน้าต่างที่ใช้
5. กลับไปเช็กใน `3D` และ `Section`

### Checkpoint หลัง Split Rebar

1. จำนวนชุดเหล็กเพิ่มขึ้นตามที่ตั้งใจหรือไม่
2. bar mark ยังอ่านความหมายได้หรือไม่
3. ไม่มีเส้นซ้อนในตำแหน่งเดิมหรือไม่

---

## 64. Cut Rebar

ใช้ตัดปลายเหล็กเมื่อชน geometry หรือเมื่อปลายเหล็กยาวเกินไป

เหมาะกับกรณี

- ปลายเหล็กชน opening
- ปลายเหล็กทะลุออกนอก host
- ต้องการให้เหล็กจบที่ตำแหน่งเฉพาะ

### วิธีใช้แบบเร็ว

1. เลือกเส้นหรือชุดเหล็กที่ปลายเกิน
2. คลิก `Cut Rebar`
3. ระบุขอบหรือหลักการตัด
4. ตรวจใน `Section` ทันทีว่าปลายเหล็กยังอยู่ใน host

### Checkpoint หลัง Cut Rebar

1. ปลายเหล็กหายจากจุดที่ชนจริงหรือไม่
2. จำนวนเส้นยังเท่าเดิมหรือไม่
3. ไม่มีเส้นใหม่เพิ่มขึ้นจากการเผลอ create ซ้ำหรือไม่

---

## 65. Rebar Set

ใช้จัดการชุดเหล็กหลังสร้างแล้ว เช่นปรับโครงสร้างของชุด หรือควบคุมรูปแบบการกระจาย

ให้คิดง่าย ๆ ว่าเป็นคำสั่งสำหรับ "จัดระบบ" ชุดเหล็ก ไม่ใช่แค่สร้างใหม่

### วิธีใช้แบบเร็ว

1. เลือก rebar set ที่ต้องการแก้
2. คลิก `Rebar Set`
3. ปรับค่าเฉพาะของชุดที่ต้องการ
4. กด `OK` หรือ `Apply` ตามหน้าต่างที่ใช้
5. ตรวจว่าจำนวนเส้นและการกระจายยังตรงกับสิ่งที่ต้องการ

### Checkpoint หลัง Rebar Set

1. `Spacing` หรือ `Quantity` เปลี่ยนไปตามที่คาดหรือไม่
2. จำนวนเส้นรวมยังสมเหตุสมผลหรือไม่
3. ถ้าจะออก `Schedule` ต่อ ให้เปิดตรวจ bar mark ก่อน

---

## 66. ลำดับการใช้งานที่แนะนำ

1. สร้างเหล็กด้วย `Main Bars` หรือคำสั่งหลักก่อน
2. เช็กผลใน `3D` และ `Section`
3. ถ้ารูปทรงหลักถูก แต่ปลายบางส่วนไม่ถูก ให้ใช้ `Cut Rebar`
4. ถ้าต้องแบ่งช่วง ให้ใช้ `Split Rebar`
5. ถ้าทั้งชุดผิด ให้ใช้ `Delete Rebar` แล้วเริ่มใหม่

> ก่อนใช้เครื่องมือ edit ทุกตัว ให้เช็กก่อนว่าโมเดลหลักถูกแล้วใน `3D` และ `Section` มิฉะนั้นจะกลายเป็นการแก้ปัญหาทั้งที่ต้นเหตุยังไม่ถูก

> [!WARNING]
> หลังใช้ `Split Rebar`, `Cut Rebar`, หรือ `Rebar Set` ให้กลับไปตรวจจำนวนเส้น, bar mark, และการกระจายใน `3D` หรือ `Schedule` ทุกครั้ง เพื่อกันเหล็กซ้อนหรือ quantity เพี้ยนโดยไม่รู้ตัว

---

## 67. หลักคิดที่สำคัญ

- ถ้าผิดทั้งระบบ ให้ลบแล้วเริ่มใหม่
- ถ้าผิดเฉพาะปลายหรือบางช่วง ให้แก้ด้วยเครื่องมือ edit
- อย่าพยายามฝืนแก้ปัญหาด้วยการใส่ค่าซับซ้อนตั้งแต่แรก
