## 75. ทำไมบทนี้สำคัญ

ถ้าคุณใช้ PowerPack แค่ทดลองทีละ element อาจยังไม่รู้สึกถึงความสำคัญของบทนี้  
แต่ถ้าจะใช้กับงานจริงทั้งโปรเจกต์หรือใช้ร่วมกันหลายคน บทนี้สำคัญมาก

เครื่องมือหลักในบทนี้คือ

- `Design Templates`
- `Design Status`
- `Configurations`

> บทนี้เป็นบท "ตั้งระบบก่อนใช้งานจริงในทีม" มากกว่าบทฝึกคลิกตามครั้งแรก ถ้าคุณยังอยู่ช่วงฝึก host ทีละตัว ให้ทำบทโมเดลหลักให้ผ่านก่อน แล้วค่อยกลับมาบทนี้

---

## 76. Design Templates

ใช้สร้างหรือเลือก template มาตรฐานสำหรับองค์ประกอบแต่ละชนิด เช่น

- beam
- column
- wall
- slab
- footing

หลักคิดคือ

- ถ้ามีองค์ประกอบคล้ายกันหลายตัว ไม่ควรตั้งค่าทีละตัวใหม่ทั้งหมด
- ควรมีแนวทางตั้งต้นที่ทีมใช้ร่วมกัน

### วิธีเริ่มใช้แบบง่ายที่สุด

1. เลือก element ต้นแบบที่ทำถูกแล้ว
2. คลิก `Design Templates`
3. สร้างหรือบันทึก template จาก element นั้น
4. ตั้งชื่อ template ให้สื่อความหมาย เช่นแยกตาม beam, wall, footing
5. ทดลองนำ template ไปใช้กับ element ที่คล้ายกัน

---

## 77. Design Status

ใช้เช็กสถานะของ element ที่เลือก ว่ากำลังอ้างอิงการตั้งค่าอะไรอยู่ หรือพร้อมสำหรับขั้นตอนถัดไปหรือไม่

เหมาะกับการใช้ตอน

- ตรวจงานก่อนคำนวณ
- ตรวจงานก่อนออก detail หรือ schedule
- เช็กว่าชิ้นต้นแบบกับชิ้นที่คัดลอกมีสภาพพร้อมใกล้เคียงกันหรือไม่

### วิธีใช้แบบเร็ว

1. เลือก element
2. คลิก `Design Status`
3. อ่านสถานะของการตั้งค่า/template/ความพร้อม
4. ถ้าสถานะยังไม่นิ่ง อย่าเพิ่ง copy หรือออกเอกสาร

## 78. Configurations

ใช้ตั้งค่าระบบของ PowerPack ในระดับโปรเจกต์หรือสภาพแวดล้อมการทำงาน

ตัวอย่างแนวคิดของสิ่งที่ควรคุมให้สม่ำเสมอ

- พฤติกรรมของคำสั่ง
- รูปแบบการแสดงผล
- การตั้งค่าเอกสาร
- รูปแบบผลลัพธ์

### วิธีใช้แบบเร็ว

1. คลิก `Configurations`
2. ตั้งค่ามาตรฐานของโปรเจกต์ก่อนเริ่มงานจริง
3. บันทึกหรือยืนยันค่าที่ทีมจะใช้ร่วมกัน
4. ทดสอบกับ element ต้นแบบ 1 ตัวก่อนขยายไปทั้งโปรเจกต์

---

## 79. วิธีใช้ในงานจริง

1. ตั้ง `Configurations` ก่อนเริ่มงาน
2. สร้าง `Design Templates` สำหรับงานที่ใช้บ่อย
3. เริ่มทำ element ต้นแบบ
4. ใช้ `Design Status` เช็กก่อนขยายงานไปยัง element อื่น

---

## 80. สรุปแบบสั้นที่สุด

- `Configurations` = ตั้งระบบ
- `Design Templates` = ตั้งมาตรฐาน
- `Design Status` = เช็กความพร้อมของ element
