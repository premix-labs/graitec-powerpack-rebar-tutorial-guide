## 7. แบบฝึกหัดที่ 2: คัดลอกเหล็กจากคานตัวแรกไปคานตัวที่สอง

หลังจากทำคานตัวแรกสำเร็จ ให้ฝึกคำสั่งนี้ทันที

### 7.1 เตรียมคานปลายทาง

1. กลับไปที่แท็บ `Structure`
2. ใช้คำสั่ง `Beam`
3. วางคานอีก 1 ตัว โดยให้ขนาดใกล้เคียงกับคานตัวแรก
4. กด `Esc`

> [!WARNING]
> ก่อนใช้ `Copy Rebar` ให้เช็กว่าคานปลายทางมีชนิด host, หน้าตัด, แนววาง, และ `Rebar Cover` ใกล้เคียงกับคานต้นแบบจริง มิฉะนั้นคำสั่งอาจทำงานผ่าน แต่ผลลัพธ์จะผิดด้านหรือผิดระยะโดยไม่ชัดเจนทันที

### 7.2 เรียกคำสั่ง Copy Rebar

1. ไปที่แท็บ `PowerPack Rebar`
2. คลิก `Copy Rebar`

### 7.3 ลำดับการกดที่ถูกต้องของ Copy Rebar

คำสั่งนี้มีหลายช่วง และผู้เริ่มต้นมักงงเพราะต้องกด `Finish` มากกว่า 1 ครั้ง

ลำดับที่ถูกต้องคือ

1. เลือกคานต้นแบบที่มีเหล็กแล้ว
2. กด `Copy Rebar`
3. โปรแกรมจะเข้าสู่ขั้นเลือกชุดเหล็ก
4. คลิกเลือกเหล็กบนคานต้นแบบก่อน
5. ถ้าต้องการคัดลอกทั้งหมด ให้เลือกเหล็กหลักและปลอกให้ครบ หรือถ้าโปรแกรมอยู่ในโหมด exclude ให้ไม่ต้อง exclude อะไรเพิ่ม
6. กด `Finish` ครั้งที่ 1
7. โปรแกรมจะเข้าสู่ขั้นเลือกคานปลายทาง
8. คลิกคานปลายทาง
9. กด `Finish` ครั้งที่ 2 ถ้าคำสั่งยังรอการยืนยัน

### 7.3.1 จำแบบสั้นที่สุด

`คลิกเหล็กก่อน -> Finish -> คลิกคานปลายทาง -> Finish`

### 7.4 ถ้าทำไมคลิกคานไม่ได้

ถ้าคุณอยู่ในคำสั่ง `Copy Rebar` แล้วคลิกคานไม่ได้ สาเหตุส่วนใหญ่คือคุณยังอยู่ในขั้นเลือกเหล็กอยู่

สัญญาณที่บอกว่าอยู่ผิดขั้น เช่น

- ด้านล่างขึ้นข้อความประมาณ `Select reinforcements to exclude from copying`

แปลว่าโปรแกรมกำลังให้เลือก `rebar` ไม่ใช่ `beam`

วิธีแก้คือ

1. คลิกเลือกเหล็กบนคานต้นแบบก่อนให้ครบ
2. กด `Finish`
3. แล้วค่อยคลิกคานปลายทางในขั้นถัดไป
4. ถ้าคำสั่งยังไม่จบ ให้กด `Finish` อีกครั้ง

### 7.5 เช็กผล

1. เปิด `3D View`
2. ซูมไปที่คานตัวใหม่
3. ตรวจว่าเหล็กถูกคัดลอกมาครบ
4. เปิด `Section` ผ่านคานปลายทาง แล้วตรวจว่าเหล็กไม่กลับด้านและไม่ทะลุออกนอก host

### ถ้าผลไม่สวย

ให้เช็กว่า

- คานสองตัวมีขนาดใกล้กันหรือไม่
- `Rebar Cover` เหมือนกันหรือไม่
- แนวการวางคานต่างกันมากหรือไม่
- คานปลายทางเป็น mirrored หรือกลับ orientation หรือไม่

ถ้าต่างกันมาก ให้สร้างใหม่แทนการ copy

### สรุปสั้นที่สุดของ Copy Rebar

`Finish` ครั้งแรก = จบการเลือกเหล็กบนคานต้นแบบ

`Finish` ครั้งที่สอง = จบการเลือกคานปลายทาง
