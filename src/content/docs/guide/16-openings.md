## 56. ใช้ Openings ตอนไหน

ใช้เมื่อ host มีช่องเปิดที่ต้องเสริมเหล็กรอบขอบ เช่น

- ช่องบันได
- ช่องลิฟต์
- ช่องงานระบบ
- ช่องเปิดในผนังคอนกรีต

---

## 57. เตรียม model ก่อน

1. ต้องมี `Opening` อยู่ใน host ก่อน
2. host ต้องเป็นคอนกรีต
3. host ต้องเป็น `Structural` ถ้างานนั้นต้องใช้คุณสมบัติโครงสร้าง
4. เปิด `Plan`, `Section`, และ `3D` ไว้ตรวจผล

> [!IMPORTANT]
> `Opening` ต้องเป็นช่องจริงที่ตัด host แล้ว ไม่ใช่แค่สัญลักษณ์, family, หรือ void ที่ยังไม่ตัด host จริง ถ้ายังไม่แน่ใจ ให้เช็กใน `Section` ก่อนกด `Openings`

---

## 58. ขั้นตอนพื้นฐาน

1. เลือก host ที่มี opening
2. ไปที่แท็บ `PowerPack Rebar`
3. คลิก `Openings`
4. ในหน้าต่างของคำสั่ง ให้เลือกชนิดเหล็กที่จะเสริมรอบขอบ opening
5. ตั้งค่า cover หรือ offset ให้สอดคล้องกับ host
6. กด `OK` หรือ `Apply`
7. กลับไปตรวจใน `Plan`
8. ตรวจต่อใน `Section` และ `3D`

> สำหรับการฝึกครั้งแรก ให้ใช้ opening สี่เหลี่ยมธรรมดา และอย่าเริ่มจาก host ที่มี opening หลายช่องในชิ้นเดียว

### 58.1 วิธีทำที่ปลอดภัยที่สุด

1. ทดลองกับ `Slab` หรือ `Wall` ที่มี opening เดียวก่อน
2. สร้างเหล็กหลักของ host ให้เรียบร้อยก่อน
3. ค่อยเรียก `Openings` เพื่อเสริมรอบช่องเปิด
4. ถ้าเหล็กรอบ opening ชนเหล็กหลักมากเกินไป ให้กลับไปเช็ก `Rebar Cover` และ offset ก่อนแก้ spacing

> [!WARNING]
> อย่าเริ่มใช้ `Openings` กับ host ที่มีหลาย opening หรือ geometry ซับซ้อนตั้งแต่ครั้งแรก เพราะถ้าผลลัพธ์ผิดจะย้อนหาสาเหตุได้ยากกว่า opening เดี่ยวมาก

### 58.2 ลำดับกดแบบใช้งานจริง

1. คลิก host ที่มี opening
2. ตรวจใน `Properties` ว่า host ยังเป็นคอนกรีตและเป็น `Structural`
3. คลิก `Openings`
4. เลือกชุดเหล็กหรือชนิดเหล็กที่ต้องการใช้เสริมรอบ opening
5. ตั้งค่า cover/offset
6. กด `OK`
7. เปิด `Plan` ดูว่ารอบ opening มีเหล็กครบทุกด้าน
8. เปิด `Section` ดูว่าเหล็กยังอยู่ในเนื้อคอนกรีต

### 58.3 วิธีเช็กว่า opening เป็นช่องจริงใน host

1. เปิด `Section`
2. มองให้เห็นขอบ opening ผ่านความหนาของ host
3. ถ้าเห็นเพียงสัญลักษณ์หรือเส้น 2D แต่ไม่เห็นช่องตัดใน host จริง อย่าเพิ่งกด `Openings`
4. ให้กลับไปแก้ opening ให้ตัด host จริงก่อน

---

## 59. วิธีตรวจผล

- ดูใน `Plan` ว่าเหล็กรอบ opening ต่อเนื่องครบทุกด้าน
- ดูใน `Section` ว่าเหล็กไม่ทะลุออกนอก host
- ดูใน `3D` ว่าเหล็กไม่ชนกับเหล็กหลักเกินจำเป็น

---

## 60. ปัญหาที่เจอบ่อย

- opening ยังไม่ได้ถูกสร้างเป็นช่องจริงใน host
- cover กับ offset ไม่ตรงกัน
- host ซับซ้อนเกินไปสำหรับการฝึกครั้งแรก
- มีการกด `Openings` ก่อนที่ host จะมีเหล็กหลัก จึงอ่านผลลัพธ์ยาก

ถ้าจะฝึกครั้งแรก ให้ใช้ opening สี่เหลี่ยมธรรมดาใน slab หรือ wall ก่อน
