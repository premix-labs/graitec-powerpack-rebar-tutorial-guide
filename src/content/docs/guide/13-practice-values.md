## 33. หน้านี้ใช้ทำอะไร

หน้านี้รวบรวม "ค่าตัวอย่างเพื่อฝึก workflow" สำหรับเปิดดูเร็วเวลาต้องการลองกดในโปรแกรมทันที โดยไม่ต้องเปิดหาค่าจากหลายบท

> ค่าทั้งหมดในบทนี้เป็นค่าตัวอย่างสำหรับการฝึกใช้งานโปรแกรม ไม่ใช่ค่ามาตรฐานตายตัวสำหรับทุกโครงการ

---

## 34. ค่าตัวอย่างงานคาน

### คานตัวอย่างพื้นฐาน

- `Bottom Longitudinal` = `2 x 10M`
- `Top Longitudinal` = `2 x 10M`
- `Transversal Bars` = `10M @ 150 mm`
- `Cover` = `40 mm`

### ใช้ตอนฝึกอะไร

- ฝึก `Main Bars`
- ฝึก `Rebar Visibility`
- ฝึก `Copy Rebar`

---

## 35. ค่าตัวอย่างงานเสา

### เสาตัวอย่างพื้นฐาน

- `Main Vertical Bars` = `8 x 10M`
- `Ties/Stirrups` = `10M @ 150 mm`
- `Cover` = `40 mm`

### ใช้ตอนฝึกอะไร

- ฝึก `Main Bars`
- ฝึกดูเหล็กใน `Section`
- ฝึก starter bars และ lap splice ต่อในบทเสา

---

## 36. ค่าตัวอย่างงานพื้น

### พื้นตัวอย่างพื้นฐาน

- `Direction 1` = `10M @ 150 mm`
- `Direction 2` = `10M @ 150 mm`
- `Layer` = `Bottom` ก่อน แล้วค่อย `Top`
- `Cover` = `25 mm` หรือค่าที่ตั้งไว้ในพื้น

### ใช้ตอนฝึกอะไร

- ฝึก `Auto Fabrics for Slabs`
- ฝึกดูผลใน `Plan` และ `3D`

---

## 37. ค่าตัวอย่างงานผนัง

### ผนังตัวอย่างพื้นฐาน

- `Vertical Bars` = `10M @ 200 mm`
- `Horizontal Bars` = `10M @ 200 mm`
- `Cover` = `40 mm`
- `Offset` = `40 mm` ทุกด้าน
- `Hook` = `0`

### ใช้ตอนฝึกอะไร

- ฝึก `Main Bars`
- ฝึกแก้ warning เรื่องเหล็กออกนอก host
- ฝึกดูผลใน `Section` และ `3D`

---

## 38. ถ้าโปรแกรมคำนวณ spacing ไม่ตรงเลขที่พิมพ์

ตัวอย่างเช่น

- คุณใส่ `200 mm`
- โปรแกรมแสดง `197 mm` หรือ `195 mm`

ถือเป็นเรื่องปกติในหลายกรณี เพราะปลั๊กอินกำลังกระจายเหล็กตามระยะใช้งานจริงหลังหัก cover และ offset แล้ว

หลักการคือ

- ถ้าผลลัพธ์ยังใกล้เคียงค่าที่ต้องการ ให้ถือว่าใช้ได้ในการฝึก
- ถ้างานจริงต้องการ `@200` เป๊ะ ให้ตรวจวิธีตั้ง distribution และ quantity เพิ่มอีกครั้ง

## 39. สรุปแบบเปิดดูเร็ว

| งาน | ค่าเริ่มต้นแนะนำ |
| --- | --- |
| คาน | `2-2 10M`, ปลอก `10M @ 150` |
| เสา | `8-10M`, ปลอก `10M @ 150` |
| พื้น | `10M @ 150` สองทิศ |
| ผนัง | `10M @ 200` สองทิศ |

---

## 40. วิธีใช้บทนี้ให้คุ้มที่สุด

1. เปิดบทนี้ไว้ข้างโปรแกรม
2. เลือก host ให้ถูก
3. ใช้ค่าตัวอย่างเพื่อให้เห็นผลเร็ว
4. เมื่อขึ้นผลใน 3D แล้ว ค่อยกลับไปปรับตามงานจริง
