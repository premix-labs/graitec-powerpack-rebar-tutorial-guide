## 5. แบบฝึกหัดก่อนเริ่มจริง: สร้างโมเดลตัวอย่าง 6 x 6 ช่อง

ก่อนใส่เหล็ก ให้เริ่มจากโมเดลตัวอย่างมาตรฐานก่อน เพื่อให้ทุกคนฝึกตามได้เหมือนกัน

เป้าหมายของโมเดลตัวอย่างนี้คือ

- มี `Grid` 6 เส้นแนวตั้ง
- มี `Grid` 6 เส้นแนวนอน
- ระยะห่างทุกช่วง `6000 mm`
- วางเสาด้วยคำสั่ง `At Grids`
- วางคานด้วยคำสั่ง `On Grids`

เมื่อทำครบแล้ว คุณจะได้โครงสร้างกริด 5 x 5 ช่อง พร้อมสำหรับฝึกงาน rebar

### 5.1 สร้าง Grid เส้นแรก

1. ไปที่แท็บ `Structure`
2. มองหากลุ่ม `Datum`
3. คลิกคำสั่ง `Grid`
4. คลิกจุดเริ่มต้นด้านซ้ายของพื้นที่ทำงาน
5. คลิกจุดปลายด้านขวาเพื่อวางเส้น grid แนวนอน 1 เส้น
6. กด `Esc`

### 5.2 คัดลอก Grid ให้ครบ 6 เส้นแนวนอน

1. คลิกเลือกเส้น grid แนวนอนที่เพิ่งสร้าง
2. ไปที่แท็บ `Modify`
3. คลิกคำสั่ง `Copy`
4. ที่แถบตัวเลือกด้านบน ให้ติ๊ก `Multiple`
5. คลิกจุดอ้างอิงบนเส้น grid เดิม
6. เลื่อนเมาส์ขึ้นหรือลงในแนวดิ่ง
7. พิมพ์ระยะ `6000`
8. กด `Enter`
9. คลิกซ้ำจนครบ 5 ครั้ง เพื่อให้ได้ grid แนวนอนรวมทั้งหมด 6 เส้น
10. กด `Esc`

### 5.3 สร้าง Grid แนวตั้งให้ครบ 6 เส้น

ทำแบบเดียวกันกับแนวนอน แต่หมุนทิศทางเป็นแนวตั้ง

1. คลิก `Grid`
2. วางเส้น grid แนวตั้ง 1 เส้นจากล่างขึ้นบน
3. กด `Esc`
4. เลือกเส้น grid แนวตั้งนั้น
5. คลิก `Copy`
6. ติ๊ก `Multiple`
7. คลิกจุดอ้างอิงบนเส้นเดิม
8. เลื่อนเมาส์ไปด้านขวาหรือซ้าย
9. พิมพ์ `6000`
10. กด `Enter`
11. คลิกซ้ำจนครบ 5 ครั้ง เพื่อให้ได้ grid แนวตั้งรวมทั้งหมด 6 เส้น
12. กด `Esc`

### สิ่งที่ควรเห็นหลังสร้าง Grid

- มีเส้น grid แนวนอน 6 เส้น
- มีเส้น grid แนวตั้ง 6 เส้น
- ระยะช่องเท่ากันทุกช่อง
- ได้ผัง grid รวมเป็น 5 x 5 ช่อง

### 5.4 วางเสาด้วย At Grids

1. ไปที่แท็บ `Structure`
2. คลิกคำสั่ง `Column`
3. ดูที่แถบตัวเลือกด้านบนของคำสั่ง column
4. เลือกโหมด `At Grids`
5. ดูที่ `Properties` แล้วเลือกชนิดเสาคอนกรีตที่ต้องการ
6. ลากเมาส์ครอบพื้นที่ grid ทั้งหมด หรือคลิกเลือกจุดตัด grid ตามต้องการ
7. กด `Finish` หรือกดยืนยันตามคำสั่ง
8. กด `Esc`

### สิ่งที่ควรเห็นหลังวางเสา

- มีเสาอยู่ที่ทุกจุดตัดของ grid
- เสาวางตรงตำแหน่ง grid พอดี
- ผังดูเป็นระเบียบและตรวจง่าย

### 5.5 วางคานด้วย On Grids

1. ไปที่แท็บ `Structure`
2. คลิกคำสั่ง `Beam`
3. ดูที่แถบตัวเลือกด้านบน
4. เลือกโหมด `On Grids`
5. ดูที่ `Properties` แล้วเลือกชนิดคานคอนกรีต
6. คลิกหรือครอบตามแนว grid ที่ต้องการให้มีคาน
7. วางคานให้ครบแนวรอบ ๆ และแนวภายใน grid
8. กด `Esc`

### สิ่งที่ควรเห็นหลังวางคาน

- คานวิ่งตามแนว grid
- ปลายคานชนกับเสาที่จุดตัด grid
- โมเดลเริ่มเป็นกรอบโครงสร้างชัดเจน

### 5.6 เปิด 3D เพื่อตรวจโมเดล

1. ไปที่ `Project Browser`
2. ดับเบิลคลิก `{3D}`
3. หมุนดูโมเดล

### สิ่งที่ควรเห็นใน 3D

- Grid เป็นระเบียบ
- เสาวางครบตามจุดตัด
- คานเชื่อมต่อกันตามแนว grid

### 5.7 ถ้ายังวางไม่ตรง ให้เช็ก 3 จุดนี้

1. ตอนวางเสา ใช้โหมด `At Grids` จริงหรือไม่
2. ตอนวางคาน ใช้โหมด `On Grids` จริงหรือไม่
3. ระยะ copy ของ grid เป็น `6000 mm` ทุกครั้งหรือไม่

เมื่อโมเดลตัวอย่างนี้พร้อมแล้ว ค่อยเริ่มใส่ `Main Bars` ให้คานในบทถัดไป
